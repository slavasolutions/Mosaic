The `content-minimal/` folder contains no `tokens/` collection. The Mosaic web profile is still respected — pages live under `pages/` and route by filename — but the design-tokens profile is absent.

Switching to this variant in the explore picker swaps the entire visual language in the URL. Same adapter, same code path, different folder. That is the format-agnostic promise: the folder is the website, and the folder can leave parts out.
