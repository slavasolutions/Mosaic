Same spec, every shape, both adapters.

Each row below is a content shape — a single page, a small blog, or a ten-plus-page site — rendered twice: once by the Astro adapter and once by the Next adapter. Same Mosaic folder, same records, same URLs. The page bytes differ because the framework differs; everything underneath is identical.
