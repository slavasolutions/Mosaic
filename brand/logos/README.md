# Mosaic Logo Variants

All logos: 24×24 viewBox, single `<svg>` element, pastel palette only, no JS.

## Recommended

**`headline.svg`** — the pick for the explainer site and topbar. One large dominant mint tile anchors the top-left; varied satellite chips of different sizes and all palette colors surround it. Reads clearly at 20 px (topbar) and 40 px (hero). Captures the "real mosaic" energy the brand needs — no uniform grid.

## All Variants

| File | Description | Palette lean |
|------|-------------|--------------|
| `grid-3x3.svg` | Original 3×3 uniform grid. Nine equal 6×6 tiles. | Balanced |
| `headline.svg` | **Recommended.** Large anchor tile + mixed-size satellites. | Balanced |
| `chips-scattered.svg` | Loose scatter: one large tile top-left, many small chips at irregular positions. | Balanced |

| `cluster.svg` | One large tile + warm chip cluster — yellow/peach/coral. | Warm |
| `cool-asymmetric.svg` | Staircase layout in mint/blue/lavender. | Cool |
| `two-column.svg` | Newspaper-style: wide left column, narrow right column of 4 tiles. | Balanced |
| `micro-grid.svg` | Dense 5×5 grid with 3 tiles intentionally absent for visual rhythm. | Balanced |
