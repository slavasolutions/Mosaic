# §4. Conformance

A directory tree is a **conforming Mosaic folder** if every record,
collection, and name within it satisfies §5–§9. A tool is a **conforming
consumer** if it resolves identity, collections, and sidecars as specified
and preserves unknown fields (`ref:format/unknown-fields`).

A specification that builds on this one MAY add rules but MUST NOT contradict
the sections below.
